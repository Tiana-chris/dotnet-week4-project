# Student Management System

Name: Amaka Christiana

## Project Description
This is a C# console application that manages stsudent records. The system allows users to add students, view records, calaculate grades, update student information, and display statistics.

## How to Run the Program
1. Open the file in Visual Studio Code.
2. Run the program.
3. Use the menu options to manage student data.

## Features
- Add new students
- View all students
- Calculate average grade
- Find student by ID
- Update student grade
- Delete student
- Display statistics
